# ☯ Zhou Divinity Technique

### *An Elegant Dialogue with the Ancient Records of Change*

[![Deploy to GitHub Pages](https://github.com/UnTamed-Fury/zhou-divinity/actions/workflows/deploy.yml/badge.svg)](https://github.com/UnTamed-Fury/zhou-divinity/actions/workflows/deploy.yml)

The **Zhou Divinity Technique** is a premium, scholarly digital implementation of the **I Ching (Yijing / Book of Changes)**. Designed with the "Ink & Echo" aesthetic, this application provides a meditative space to explore the 3,000-year-old philosophical system that maps the flow of existence through binary geometry.

---

## ✨ Key Features

-   **Dual Divination Methods:**
    -   **Traditional Yarrow Stalk:** A meditative process adhering to classic Zhou Dynasty probabilities (1/16, 5/16, 7/16, 3/16).
    -   **Modern Three-Coin:** A direct and immediate technique for rapid insight.
-   **SVG Brush-Stroke Rendering:** Hexagrams are dynamically drawn using hand-crafted SVG paths with a unique "ink-bleed" animation.
-   **Comprehensive Interpretation:** Complete 64-hexagram database based on the authoritative **Richard Wilhelm & Cary F. Baynes** translation.
-   **Intelligent Logic:** 
    -   Automatic identification of **Changing Lines**.
    -   Dynamic mapping to **Relating (Future) Hexagrams**.
    -   Trigram visualization and breakdown (Above/Below).
-   **Record of Changes:** LocalStorage-based history to track and recall your recent consultations.
-   **Responsive Scholarly Design:** A fluid, "intentional" UI that scales perfectly from mobile to ultra-wide displays.

## 🚀 Local Development

Ensure you have [Yarn](https://yarnpkg.com/) installed.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/UnTamed-Fury/zhou-divinity.git
    cd zhou-divinity
    ```

2.  **Install dependencies:**
    ```bash
    yarn install
    ```

3.  **Start development server:**
    ```bash
    yarn dev
    ```

4.  **Build for production:**
    ```bash
    yarn build
    ```

5.  **Preview production build:**
    ```bash
    yarn preview
    ```
